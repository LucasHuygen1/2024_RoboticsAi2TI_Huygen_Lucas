# Lidar Obstakelvermijding met ROS2

de laser_callback functie wordt elke keer aangeroepen als er een nieuwe Lidar gegeven binnenkomt. het gaat berekenen waar obstakels zijn aan de voorkant links en rechts

in de motion functie gebeuren de beslissingen.
het gaat bepalen wanneer hij naar links, rechts of moet rotaten. dmv de sensor gegevens.
